# AI-Powered Fake Profile Detector

**Live Demo:** [Click Here](https://fake-profile-detector-yourusername.streamlit.app)

---

## 🔍 Project Description

This is a **web-based AI application** that detects whether a social media profile is **real or fake** using machine learning. The system analyzes key features like **followers, following count, posts, and bio length** to predict suspicious or bot-like profiles.

Built with **Streamlit** for the frontend and **scikit-learn** for the ML model, the app provides:

- Real-time predictions  
- Probability scores for fake profiles  
- Explanation of why a profile is considered fake  
- History of past checks stored in a database  

This project demonstrates **practical AI application for online safety** and can be deployed live for easy access.
